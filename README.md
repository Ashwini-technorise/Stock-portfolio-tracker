# Stock-Portfolio Tracker

## Description
The stock portfolio tracker is a simple application that helps users manage and track their stock investments. It allows users to add stocks, monitor their portfolio, and calculate profit or loss.

## Features 
- Add and manage stock investments
- Track quantity and price of stocks
- Calculate total investment value
- View profit or loss
- User-friendly interface

## Tchnologies used
- Python
- Basic Data Structures (list, dictionary)

## How to run
1. Open terminal
2. Run the program: Python task2.py
